INLINE ELEMENTS

  • #a - Creates a link to a URL: a web page, a section within a page, an email address
  • #span - Defines a generic inline container of content, that does not carry any semantic value.
  • #var - Defines a variable in a mathematical or programming expression.
  • #u - Makes an element's text underlined.
  • #wbr - Defines a location within a block of text where the browser could eventually insert line break.

BLOCK ELEMENTS

  • #div - Defines a generic block of content, that does not carry any semantic value. Used for layout elements like containers.
  • #head - Defines a container for a web page's metadata.
  • #header - Defines the header of a web page or section.
  • #iframe - Defines a container for a nested browsing context: you can include a web page within another web page.
  • #map - Defines an interactive map over an image.
